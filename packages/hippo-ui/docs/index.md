# HippoUI
### 说明
- Vue3、Vite3、pnpm、ESBuild
- TypeScript support
- Unit Testing whit Vitest
- Eslint+Prittier
- UnoCss- the instant on-demand atomic CSS engine
- 🚘 CI/CD with GithubActions

### Install
---
```js
npm i hippo-ui
```

### Quick Start
```js
import Vue from 'vue'
import HippoUI form 'hippo-ui'
const App = {
    template:`
    <HButton />
    `
};
createApp(App).use(HippoUI).mount("#app");

```
